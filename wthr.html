Applying LLMs to improve how we receive weather forecasts.

GPT-3 API Prompt Engineering Met Office API

Sep 2022 - Present

How might we make the weather forecast more user-friendly? All you really want to know is if the weather will be different from yesterday. Or whether you should wear a coat. That's exactly what Ben and I set out to create by manually crafting short summaries based on the Met Office API before LLMs took off.

We soon discovered that GPT3 does a better job. Through continuous improvements, we have created effective prompts that mitigate LLMs from ‘hallucinating’ inaccurate or misleading forecasts.

But that's not all. In addition to providing useful weather advice, we collaborated with ChatGPT to inject some creativity and fun into our widget. With over 200 styles to choose from, our weather forecasts range from thought-provoking to downright silly.
